# OpenMoji Animal Icons

These SVG icons are sourced from the OpenMoji project and copied into this repo
for use in the Fantasy Oscars UI.

- Source: OpenMoji (https://openmoji.org/)
- License: Creative Commons Attribution-ShareAlike 4.0 International (CC BY-SA 4.0)

Notes:

- Files are taken from the `openmoji-svg-black` distribution and renamed to the
  app's internal animal identifiers (e.g. `monkey.svg`, `gorilla.svg`).
